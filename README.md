# Guía Rápida BAPE SACYL

## Descripción
Sitio web estático e interactivo que proporciona información completa sobre la Bolsa Abierta y Permanente (BAPE) del Servicio de Salud de Castilla y León (SACYL).

## Contenido
El sitio web incluye cuatro módulos principales:

1. **ACCESO BAPE**: Fundamentos y proceso de inscripción
2. **PUNTUACIÓN Y ESTRATEGIA**: Baremo de méritos con simulador interactivo
3. **LLAMAMIENTOS**: Tipos de nombramiento y estrategia de disponibilidad
4. **PENALIZACIONES**: Régimen sancionador y causas justificadas

## Características
- ✅ Diseño responsive (adaptable a móviles, tablets y escritorio)
- ✅ Navegación por pestañas con URLs amigables
- ✅ Simulador interactivo de puntuación
- ✅ Colores corporativos SACYL (azul/blanco)
- ✅ Información de contacto de UGT Sanidad Salamanca
- ✅ Enlaces a recursos oficiales
- ✅ Accesibilidad mejorada
- ✅ Botón de volver arriba
- ✅ Almacenamiento local de datos del simulador

## Estructura de Archivos
```
bape-sacyl/

├── styles.css    # Estilos y diseño responsive

```

## Cómo Usar
1. Abrir el archivo `index.html` en cualquier navegador web moderno
2. Navegar entre las pestañas usando el menú superior
3. Utilizar el simulador en la sección "PUNTUACIÓN Y ESTRATEGIA"
4. Los datos del simulador se guardan automáticamente en el navegador
